# 💼 Job Vacancy System

Sistem Job Vacancy berbasis Laravel untuk mengelola lowongan pekerjaan dan pelamar.

---

## 🚀 Fitur Utama

- 🔐 Authentication (Login & Register)
- 🏢 Manajemen Lowongan Kerja
- 👤 Manajemen Pelamar
- 🗂️ CRUD Data Lowongan

---

## 🛠️ Teknologi yang Digunakan

### Backend
- Laravel
- PHP
- MySQL

### Frontend
- Vite
- React 
- Bootstrap / Custom CSS
